Preguntas Frecuentes

¿Mis datos se usan para entrenar el modelo?

Absolutamente NO. Al ser una instalación local, los datos no salen de tu servidor. No hay entrenamiento externo ni fugas.

¿Qué pasa si no tenemos GPUs NVIDIA?

El sistema es flexible. Podemos optimizar el modelo para ejecutarse en CPU o sugerir la infraestructura mínima necesaria.

¿Cuánto tiempo tarda la indexación?

Depende del volumen. Miles de páginas se indexan en pocos minutos, permitiendo consultas instantáneas.

¿Soportan varios idiomas?

Sí, los modelos que implementamos (como Llama 3 o Mistral) son multilingües y funcionan perfectamente en español.
